## Запуск

### Вручную

```
npm ci
```

1. Dev сервер

```
npm run dev
```

2. Сборка

```
npm run build
```

### Запуск сервера, раздающего статику с помощью Docker

Сервер доступен по адресу http://localhost:3000

```
docker-compose up --build
```
